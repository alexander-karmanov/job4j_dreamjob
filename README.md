# job4j_dreamjob

Техническое задание - проект "Работа мечты" 

Особенности приложения.

    Модели данных:
        Кандидаты: публикуют свои резюме.
        Кадровики: размещают вакансии.
    Взаимодействие:
        Кандидаты могут откликаться на вакансии.
        Кадровики могут приглашать кандидатов на открытые вакансии.

Этот проект включает в себя ключевые аспекты разработки веб-приложений, 
такие как: работа с моделями, взаимодействие между пользователями 
и структура кода на уровне приложения.
